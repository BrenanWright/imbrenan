How this page works

All of these cards are written dynamically using Javascript. The method makes a GET call to get all the entries in the Testimonials table from the database in JSON format. This JSON then populates a list in the Javascript called testimonials. This testimonials list is then fed into a foreach loop which then programmatically creates all the necessary rows, divs, paragraphs, and headings to display the testimonials that have been approved.

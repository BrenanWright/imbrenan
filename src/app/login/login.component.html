Explanation of what is going on for anyone curious

There is a POST request for your username and password going on when you hit log-in on this modal-form. The back-end checks these values using the UserManager in .NetCore and returns a request based on if your credentials are verified. If they are good credentials and they are in the database it generates a JWT security token which is stored in the localstorage of your browser.
